Class: Case

Case(caseId, caseTitle, caseCategory, caseStatus)

new Case(caseId, caseTitle, caseCategory, caseStatus)

A clickable case item component.

Displays the case's title, category, and status indicator (color-coded). When clicked, it updates the selected case context with the case ID and title.

Parameters:
Name Type Description
caseId string

The unique identifier for the case.
caseTitle string

The title or subject of the case.
caseCategory string

The category or type of the case.
caseStatus boolean

The status of the case (e.g., open or closed).
